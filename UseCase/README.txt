Cas d'utilisation : Placer pari simple

Acteurs : Parieur : Utilisateur qui souhaite placer un pari sur un événement sportif.

Description :
Le parieur sélectionne un événement sportif, un type de pari simple, le résultat qu'il souhaite parier, et le montant de sa mise en jetons.
Le système enregistre le pari et débite le compte du parieur du montant de la mise.
En cas de succès, le parieur reçoit un crédit égal à sa mise multipliée par la cote, déterminée par l'algorithme du bookmaker.

Cas d'utilisation : Placer pari avancé

Acteurs : Parieur

Description :
Le parieur sélectionne un événement sportif, un type de pari avancé, les détails spécifiques du pari (comme le score exact à une certaine période du match), et le montant de sa mise en jetons.
Le système enregistre le pari et débite le compte du parieur.
En cas de succès, le parieur reçoit un crédit basé sur la cote spécifique du pari avancé.

Cas d'utilisation : S'inscrire

Acteurs: Parieur
Description: Le parieur renseigne ses informations personnelles

Cas d'utilisation : Gestion des paramètres paris

Acteurs :Bookmaker

Description :
Le bookmaker définit les paramètres pour chaque paris, tels que les cotes, les écarts sur les scores, les limites de mise, etc.


Cas d'utilisation : Gestion des evenements sportifs

Acteurs :Bookmaker, système d'informations

Description :
Le bookmaker détermine quels événements sportifs seront proposés par le système, ainsi que les variantes de paris disponibles pour chaque événement.
Le système d'information intervient dans la mise à jour en temps réél des données sportifs


Cas d'utilisation : Recharger jetons

Acteurs : Parieur

Description :
Le parieur a la possibilité de recharger son compte en jetons.
La méthode de recharge des jetons sera définie ultérieurement par le bookmaker.



Cas d'utilisation : Consulter compte

Acteurs : Parieur

Description :
Le parieur peut consulter les informations liées à son compte (infos personnelles, solde...)
