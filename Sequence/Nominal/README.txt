Consulter compte

L'acteur (P) se connecte à son compte en ligne dans le système de paris.
L'acteur sélectionne l'option "Consulter compte" dans l'interface du système.
Le système (S) envoie une requête à la base de données (BD) pour obtenir les informations du compte.
La base de données répond en fournissant les informations demandées sur le compte.
Le système affiche ensuite ces informations sur le compte de l'utilisateur.
Le processus se termine, et l'utilisateur peut consulter les détails de son compte.



GestionEventSportif

Le bookmaker accède à la plateforme de paris en ligne.
Le bookmaker sélectionne l'option "Gestion des événements sportifs" dans l'interface.
L'interface envoie une requête de gestion des événements sportifs au système.
Le système demande aux sources de données de récupérer les informations en temps réel sur les événements sportifs.
Les sources de données fournissent les données en temps réel sur les événements sportifs au système.
Le système affiche les événements sportifs disponibles dans l'interface utilisateur.
Le bookmaker sélectionne un événement sportif spécifique à gérer dans l'interface.
L'interface envoie une requête de gestion de l'événement sportif sélectionné au système.
Le système affiche les options de gestion disponibles pour l'événement sportif sélectionné dans l'interface utilisateur.


GestionParamParis

Le bookmaker accède à la plateforme de paris en ligne.
Le bookmaker sélectionne l'option "Paramètres de paris" dans l'interface utilisateur.
L'interface envoie une demande de gestion des paramètres de paris au système.
Le système demande à la base de données de récupérer les paramètres actuels.
La base de données fournit les paramètres actuels des paris au système.
Le système affiche les paramètres actuels dans l'interface utilisateur.
Le bookmaker modifie les paramètres selon ses préférences dans l'interface.
L'interface envoie les nouveaux paramètres au système.
Le système met à jour les paramètres dans la base de données.
La base de données confirme la mise à jour des paramètres au système.
Le système affiche une confirmation de la mise à jour des paramètres dans l'interface utilisateur.
L'interface confirme la mise à jour des paramètres au parieur.


Inscription

L'acteur  accède à la plateforme de paris en ligne.
L'acteur sélectionne l'option "Inscription" dans l'interface utilisateur.
L'acteur remplit le formulaire d'inscription avec les informations requises.
L'interface d'inscription transmet les informations d'inscription au système d'inscription.
Le système d'inscription vérifie la disponibilité du nom d'utilisateur dans la base de données.
La base de données confirme au système d'inscription la disponibilité du nom d'utilisateur.
Le système d'inscription crée un nouveau compte pour l'utilisateur dans la base de données.
La base de données confirme au système d'inscription la création du compte.
Le système d'inscription crédite le compte de l'utilisateur avec un capital initial de 10 000 jetons.
La base de données confirme au système d'inscription le crédit initial du compte.
Le système d'inscription affiche une confirmation d'inscription dans l'interface utilisateur.
L'interface d'inscription confirme à l'utilisateur l'inscription réussie sur la plateforme de paris en ligne.



PlacéParisAvancé

L'acteur ( parieur) choisit un événement sportif sur lequel il souhaite parier.
Le système de paris en ligne vérifie la disponibilité de l'événement dans la base de données.
La base de données confirme au système la disponibilité de l'événement.
L'acteur sélectionne le type de pari "avancé".
Le système de paris en ligne vérifie le type de pari sélectionné.
La base de données confirme au système le type de pari sélectionné.
L'acteur entre le résultat sur lequel il souhaite parier.
L'acteur définit le montant du pari en jetons.
L'acteur soumet son ticket de pari.
Le système de paris en ligne débite le compte de l'acteur du montant du pari.
La base de données confirme au système le débit du compte.
Le système de paris en ligne calcule la cote pour le pari.
La base de données confirme au système que la cote a été calculée.
Le système de paris en ligne affiche une confirmation du pari à l'acteur.

RechargerJeton

Le parieur accède à la plateforme de paris en ligne.
Le parieur sélectionne l'option "Recharger jeton".
L'interface utilisateur envoie une requête de rechargement de jeton au système de paris.
Le système de paris traite la requête et effectue un transfert de fonds vers la banque.
En cas de succès du transfert :
La banque confirme le transfert de fonds au système de paris.
Le système de paris affiche une confirmation de rechargement de jetons à l'interface utilisateur.
